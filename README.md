# Glassmorphism Loader

Glassmorphism Loader built using HTML, CSS and a touch of JavaScript — a clean, modern loader with a frosted-glass appearance that's easy to embed and customize.

![Screenshot](Screenshot%202024-06-08%20135349.png)

## Demo
Open `index.html` in your browser to see the loader in action.

## Features
- Pure HTML/CSS with minimal JavaScript
- Glassmorphism / frosted-glass visual style
- Lightweight and easy to customize
- Works in modern browsers

## Files
- `index.html` — simple demo page
- `style.css` — styling for the loader (glass effect, animation)
- `Screenshot 2024-06-08 135349.png` — demo screenshot

## Usage
1. Clone or download the repo:

```bash
git clone https://github.com/BinaryVortex/Glassmorphism-Loader.git
```

2. Open `index.html` in your browser, or embed the loader markup and CSS into your project.

### Example
Include the markup from `index.html` in your HTML file and link `style.css`:

```html
<link rel="stylesheet" href="style.css">
<div class="loader-wrapper">
  <div class="glass-loader"></div>
</div>
```

Customize colors, blur amount, or animation timing in `style.css` to match your project.

## Customization tips
- Adjust the `backdrop-filter: blur(...)` value for stronger/weaker frost effect.
- Tweak colors, opacity, and box-shadow in `style.css` to fit your theme.
- Change animation duration to speed up or slow down the loader.
